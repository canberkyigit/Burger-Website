# Burger-Website

A burger house website built with React and Vite.
Live demo : https://canberkyigit.com/diver/

## Features

- Modern, responsive layout
- Animations (Framer Motion)
- React Router for navigation
- PWA support (Vite PWA plugin)
- Count-up and Swiper components

## Tech Stack

- **React** with **Vite**
- **React Router DOM**
- **Framer Motion** for animations
- **Swiper** for carousels
- **Vite PWA** for offline/progressive web

## Setup

```bash
git clone https://github.com/canberkyigit/Burger-Website.git
cd Burger-Website
npm install
npm run dev
```

Open the URL shown in the terminal (e.g. [http://localhost:5173](http://localhost:5173)).

## Scripts

| Command           | Description           |
| ----------------- | --------------------- |
| `npm run dev`     | Start dev server      |
| `npm run build`   | Production build      |
| `npm run preview` | Preview production build |
